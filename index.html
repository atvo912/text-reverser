Text Reverser

How to Use
Simply begin typing the text that you would like reversed into the first text box, and then the reversed text will populate in the second text box.

There is a maximum limit of 300 characters, which is displayed beneath the input text box.

A "Clear Text" button has been provided for your convenience, as well.
